# 在线使用：[https://yhgzs-111.github.io/bbdown-webui/](https://yhgzs-111.github.io/bbdown-webui/)

请先在本地运行```BBdown serve```后正确填写**BBdown**服务端口

# 简介

本项目是**BBDown**的 Web 前端界面，用于在网页中以图形化界面调用**BBdown**的下载功能

**本项目仅供个人学习研究使用，请勿用于非法用途**

## 环境准备

请确保已安装以下工具：

* **Node.js**

* **npm**

* **BBdown**

## 运行
1. 转到**BBdown**所在目录，运行以下命令
```code
bbdown serve
```
2. 运行以下命令
```code
git clone https://github.com/yhgzs-111/bbdown-webui.git
cd bbdown-webui
npm install
npm install lucide-react
npm install -D tailwindcss@3 postcss@8 autoprefixer@10
npm run dev
```
